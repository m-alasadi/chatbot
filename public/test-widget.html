هذه صفحة تجريبية لاختبار مساعد مشاريع العتبة العباسية كودجت قابل للتضمين في أي موقع

✓ الودجت مفعّل في هذه الصفحة

كيف يعمل الودجت؟

  • زر عائم في الجانب الأيسر من الشاشة
  • يفتح نافذة محادثة ذكية عند الضغط عليه
  • يتواصل مع نفس API الموجود في المشروع الأصلي
  • يعمل بدون iframe - حقن مباشر في DOM
  • متجاوب مع جميع الأحجام (Mobile & Desktop)
  • سهل التضمين - سطر واحد من الكود!

📝 كود التضمين

لتضمين الودجت في أي موقع، أضف هذا الكود قبل إغلاق تاج </body>:

<!-- AlKafeel Widget - ملف واحد فقط! --> <script src="http://localhost:3000/api/widget"></script> <script> // تهيئة الودجت AlkafeelWidget.init({ apiEndpoint: 'http://localhost:3000/api/chat/site', title: 'مساعدك في المشاريع', subtitle: 'اسأل عن مشاريع العتبة العباسية', position: 'left' }); </script>

ملاحظة: في الإنتاج، استبدل localhost:3000 بالدومين الفعلي المنشور على Vercel

مميز: ملف JavaScript واحد فقط - CSS مدمج بالكامل! ✨

⚙️ الخيارات المتاحة

يمكنك تخصيص الودجت باستخدام الخيارات التالية:

AlkafeelWidget.init({ apiEndpoint: '/api/chat/site', // رابط API title: 'مساعدك في المشاريع', // عنوان الودجت subtitle: 'اسأل عن المشاريع', // نص فرعي position: 'left', // left أو right buttonText: '💬', // رمز الزر buttonSize: '60px', // حجم الزر zIndex: 999998 // ترتيب العرض });

🚀 المميزات

  • بدون iframe - أداء أفضل وأسرع
  • تصميم عصري ومتجاوب
  • دعم كامل للغة العربية مع RTL
  • تكامل سلس مع أي موقع
  • سهولة التخصيص والتعديل
  • حجم صغير وسريع التحميل
